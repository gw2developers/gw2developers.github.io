Some forums don’t allow dynamic links for images. Here is the fix for it.

I should not that this grabs the latest published data. By published data, I mean data that I pull from the API and store. It does not pull directly from the API, but instead grabs from the API, massages it, stores it, verifies it and the publishes it.

I am currently updating about once every 15 seconds.
