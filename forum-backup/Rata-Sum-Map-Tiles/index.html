This is how I think it is:

Imagine the map of Tyria as one giant Image, call this the master image.
To make this master image of all of Tyria, someone has pasted each playable map (i.e. Queensland, Kessex Hills ect.) onto this map in is relevant location.
Then a copy of the master map has been taken for use in the game.
Also a copy of the map has been taken to make the tiles in the Tile Service.
Some one has noticed the error of the placement of Rata Sum in the copy taken for the In Game Map and adjusted it.
But the copy taken for making the tiles in the Tile Servce has not been altered.

I hope that made some sense.
