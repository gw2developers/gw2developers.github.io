IF you click on “my Account” on top, you will be able to create API keys that pull certain information from your account, depending what info you check.

Trading Post API pulls value and volume of all your current listings and buy orders and your trading history (for up to 90 days).

Characters and Inventory API pulls info about the items in your bags, bank and mat storage, so you can easily calculate their value. Not sure, what else the character API also pulls on information but i think you will eventually be able to display your build, skins etc, so you dont have to manually put it into build calculators, if you want to share it with others.

The Guild API is still in developement, I think it will make guild organisation much easier for guild leaders and will include info about members different classes, levels, builds, calculating the value of items in the guild vault, login times etc.
